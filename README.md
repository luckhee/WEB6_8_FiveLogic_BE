# 🧭 잡메이트 (JobMate)
**청소년 IT 직업 탐색 및 진로 설계 플랫폼**

청소년들이 다양한 IT 직업을 탐색하고, 진로 로드맵을 설계하며, 멘토 인터뷰 영상을 통해 현업의 경험을 간접적으로 접할 수 있도록 돕는 서비스입니다.  
사용자 중심의 기능 설계와 안정적인 서비스 운영을 목표로 개발되었습니다.

---

## 📅 프로젝트 개요
- **기간:** 2025.09 ~ 2025.10  
- **팀 구성:** Backend 5명, Frontend 1명  
- **역할:**  
  - 백엔드 팀장  
  - 게시글(커뮤니티) 기능 개발  
  - 에러 로그 수집·분석·대응 체계 구축(Sentry)

---

## 🛠 기술 스택

### Backend
- **Java 21**
- **Spring Boot**
- **MySQL**
- **JPA**
- **Sentry** (실시간 에러 수집 및 모니터링)
- **OpenAI API** (추천/진로 관련 기능 활용)
  
### Infra
- **AWS EC2**
- **Docker**

---

## ✨ 주요 기능

### 📌 1. 게시글(커뮤니티) 기능
- 게시글 작성 / 조회 / 수정 / 삭제
- 예외 상황(권한, 입력 오류 등)에 대한 안정적인 예외 처리 구조 적용

### 📌 2. 사용자 경험 기반 오류 검출 및 안정성 확보
- Sentry 기반 실시간 에러 수집
- 에러 발생 시 빠른 분류 및 팀 내 공유 프로세스 수립

---

## 🧪 문제 해결 & 성과

### 🔥 Sentry 기반 에러 분석 및 운영 안정화
- QA 단계에서 **총 259건의 에러 발생**  
- 모든 에러를 **"엔드포인트 - 입력값 - 원인"** 형태로 문서화  
- 각 담당자에게 에러 원인을 명확히 전달해 **해결 속도 단축**
- 중복 오류·근본 원인 파악이 가능해짐 → **운영 안정성 향상**

### 🧩 개선 효과
- 오류 원인 파악 시간 대폭 단축  
- 기존에 파편화되어 있던 예외 처리 구조를 체계화  
- 공통 예외 처리의 구조 개선으로 에러 재발률 감소
